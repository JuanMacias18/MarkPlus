# Markplus

Marketplace universitario donde los estudiantes pueden vender productos, servicios y conocimientos. Una iniciativa de EduAcceso. Este repositorio contiene el prototipo de la landing inicial.

## Visión

Crear un ecosistema donde los estudiantes puedan estudiar, emprender y crecer profesionalmente al mismo tiempo. Generar ingresos para estudiantes, monetizar conocimientos y habilidades, y reducir la deserción estudiantil por temas económicos.

## Stack actual

HTML + CSS estáticos. Sin frameworks ni build step. Suficiente para validar el concepto y empezar a recoger feedback de usuarios reales.

## Cómo ver el sitio localmente

Solo abre `index.html` en tu navegador. Eso es todo.
